Building Blocks was a free online quiz service where teachers could create custom programming quizzes that the students could connect to and compete with each other. Building Blocks was made for my Senior Capstone project where four of my peers and I created it over the course of one semester. During our research, we found there weren’t any resources to reinforce concepts in programming in schools, like how Kahoot does with its quizzes and Nitro Type for typing skills, so we created Building Blocks to solve that problem.

![Building Blocks Poster](/assets/img/Programming/building-blocks_poster.jpg)
*Building Blocks Poster*

Building Blocks primarily used React for its front end, which is what I worked on. I learned about React components, mutations, queries, along with integrating API calls with a PSQL database to interact with the frontend. The majority of my work was done on the pages that let you select a prebuilt quiz, and build custom quizzes.

![Building Blocks Quiz Selection](/assets/img/Programming/building-blocks_quiz-selection.jpg)
*Building Blocks Quiz Selection*

We spent a semester researching, planning, programming, and then testing our project, and we found very positive results. At the end of the semester, we unfortunately had to shut down the website as A&M’s Computer Science department was no longer funding the project. If you want to learn more, you can check it out on the [GitHub repo](https://github.com/TypeGAO/building-blocks), or this video I made talking about it in the style of an infomercial.
